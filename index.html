Hi, I'm Eric. I'm a recent graduate from Northwestern University, where I studied Industrial Engineering.

Currently, I'm a Data Scientist at Asana, working on data infrastructure.

I think free and open source software is the coolest thing. I've made some small contributions to projects like scikit-learn and Boostnote, and I'm always looking for opportunities to do more.

In my free time, I like to play string instruments, write calligraphy, and make stuff with 3D printers and code. I'm a massive fan of Discworld: GNU Terry Pratchett.

Some of the code I've written and open source contributions I've made, in no particular order.

  • sklearn.preprocessing.PowerTransformer: the Box-Cox power transform, to be released in scikit-learn 0.20. Bonus: examples in the scikit-learn manual.
  • ggphysics: using R to make fun little physics animations.
  • squirrelfinder: a website to locate squirrels at Indiana University. This is a really rough WIP for the Squirrels of IU.
  • gifmake: a little command line utility for stitching images into GIFs. It's probably got at least 4 pip installs, but I haven't checked in a while.
